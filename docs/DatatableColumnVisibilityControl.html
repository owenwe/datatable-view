Class: DatatableColumnVisibilityControl

DatatableColumnVisibilityControl

A UI control to manage a collection on/off properties. In this case a series of visibility booleans from DataTable columns.

Constructor

new DatatableColumnVisibilityControl(options, groupsopt, defaultopt)

This View controls the visibility properties of DataTable columns.
Parameters:
Name Type Attributes Description
options object The configuration options for this View instance. All of the values except for the class property are passed to this view's model constructor.
Properties
Name Type Description
columns object This is expected to be an object where each property key is the "data" value in the datatable columns config. The value of each property is expected to have these properties:
  • index - original index of column from datatable columns
  • data - the name of the column key
  • title - the descriptive label for the column
  • visible - a boolean for visibility
groups Array.<object> <optional>
 An object array where each object has a name and columns properties.
default Array.<string> <optional>
 An array of column names that are visible by default. It's best to leave this parameter alone.
options.widthOverride number <optional>
 The width of the group of button switches for each of the columns. Increasing this number allows more buttons per horizontal row, saving vertical space.
