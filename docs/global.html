get() → {object}

Returns the value of the form input. This is where the valueOnly property in the data configuration comes into play. When true, the value returned from .get() is only the value from the input control. e.g. f.get() would return 1, or 'foo'. Where when valueOnly is false, or doesn't exists (default), the value returned from .get() is an object with the data configuration name property as the key and the input control value as the value, e.g. f.get() = {"foo":1}
