ButtonGroupToggleFormInput

This form control is for a value that belong to a small set of options.

Constructor

new ButtonGroupToggleFormInput(options)

Button group toggle input form control, this will create an instance of ButtonGroupToggleFormInput
Parameters:
Name Type Description
options object configuration options for this View instance
Properties
Name Type Attributes Default Description
name string the value to assign the name attribute on the input
label string the display label for the form input
buttons array an array of objects used to populate the button group, each item in the array should have this form: {label: , value: }
valueKey string a string that will be used in the .set() function to get the named value from the data object
inputAttributes object <optional>
 {autocomplete:'off'} the attributes for the input elements
defaultToggled number <optional>
 0 the index in the buttons array that should be toggled by default
disabled boolean <optional>
 false if true then the form control is initially put into a disabled state
readonly boolean <optional>
 false if true then the form control is initially put into a readonly state, note if the disabled option is true, then that takes priority
buttonGroupSize string <optional>
 the bootstrap button group size css class
orientation LAYOUT_ORIENTATION <optional>
